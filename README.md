# WarNet-Satelital
WarNet Satelital es una Nave no tripulada preparada para la Orbitacion Satelital Terrestre fuera de la estratosfera en condiciones Espaciales, con facultad de corregir su órbita y realizar desplazamientos autónomos o programados desde puestos de mando. Con recepciones y emisiones de señales es un repetidor de largo alcance, centro de datos espacial
