# WarNet-Satelital
WarNet Satelital es una Nave no tripulada preparada para la Orbitacion Satelital Terrestre fuera de la estratosfera en condiciones Espaciales, con facultad de corregir su órbita y realizar desplazamientos autónomos o programados desde puestos de mando. Con recepciones y emisiones de señales es un repetidor de largo alcance y centro de procesamiento de datos espacial autonomo, independiente y conectado.
Con facultad superior de AGI, de Nave Espacial, de Acciones y Emision de Señales Avanzadas Satelitales.
